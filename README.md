## Juego de preguntas y resuestas

Este es un juego de preguntas y respuestas, donde se verán un total de 25 preguntas,
estas serán divididas en 5 grupos de 5 niveles, al final de los niveles, este mostrará el puntaje final.

Cabe recalcar que este juego fue creado con JavaScript y se realizó por medio de POO (Programación orientada a Objetos).
